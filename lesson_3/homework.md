# Домашнее задание

Seaborn - мощная библиотека для визуализации данных. Изучите ее базовый функционал и выведите 
график линейной регрессии по данным собранным в предыдущем задании (ось X - время появления 
новости, ось Y - номер новости по возрастанию, начиная с 0). Подсказку можно найти в проверочном 
скрипте окружения (test_installation.py).

Пример данных

| Timestamp(x)          | News(Y) |
|-----------------------|---------|
| 1485331000            |  0      |
| 1485332000            |  1      |
| 1485333000            |  2      |
| 1485334000            |  3      |
| 1485335000            |  4      |
| 1485336000            |  5      |


Подумайте над следующими вопросами:

1. О чем говорит наклон графика линейной модели? 
2. Как он соотносится с результатами полученными вами в предыдущем домашнем задании?
3. Хорошо ли линейная аппроксимация подходит для ваших данных или видны систематические отклонения?
